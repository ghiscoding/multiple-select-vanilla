Lazy Load Data Code html | ts

Use { lazyData: (resolve, reject) }to lazy load data, the 2 arguments are callback functions to call when the data is ready. Lazy data will only be loaded once and reopening the dropdown will show instantly. If a failure happened and the reject callback is called, it will be retried after you close and reopen the dropdown. For testing purposes, you can refresh the page or click the button to retest the lazy loading.
Note: any pre-selected options will only show up after the data is fully loaded (e.g. 4th select with "Filter & OK button").
